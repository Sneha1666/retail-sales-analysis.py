# 🛒 Real-World Data Project: Retail Sales Analysis

## 📌 Project Overview
This project is a real-world data analysis on a retail sales dataset.  
It helps to understand how data science is used in business decisions like sales, profit, and performance tracking.

---

## 🎯 Objective
- Analyze retail sales data
- Find best performing regions
- Study profit by category
- Understand sales trends over time
- Visualize key business insights

---

## 📂 Dataset Information
The dataset includes:

- Order Date
- Region
- Category
- Sales
- Profit

This is a sample dataset designed for learning real-world analytics.

---

## ⚙️ Tools & Libraries Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

## 📊 Steps Performed

### 1. Data Creation
Created a structured retail dataset using pandas.

### 2. Data Exploration
- Checked dataset structure
- Viewed first rows
- Checked missing values
- Generated statistical summary

### 3. Data Analysis
- Sales grouped by Region
- Profit grouped by Category
- Monthly sales trend analysis

### 4. Data Visualization
- Bar chart → Sales by Region
- Bar chart → Profit by Category
- Line chart → Sales trend over time
- Scatter plot → Sales vs Profit
- Heatmap → Correlation analysis

---

## 📈 Key Insights
- Sales vary across different regions
- Technology category gives higher profit
- Sales and profit are positively related
- Sales show changes over time (monthly trend)
